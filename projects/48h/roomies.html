Roomies

Personal

Unity (C#)

Jan 2019

2 developers, 5 artists, 3 game designer
Made during the official Global GameJam PoliMI.

Cooperative game in which four players, using the same keyboard, must coordinate themself to carry out some daily chores: washing the dishes, filling the refrigerator, repairing the television, cleaning the toilet, cooking and so on.
